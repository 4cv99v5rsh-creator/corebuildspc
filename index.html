🧩 PC Parts Explained

CPU (Processor)

The brain of your PC. It handles all calculations and tasks.

GPU (Graphics Card)

Handles games, videos, and graphics performance.

RAM

Temporary memory that helps your PC run multiple tasks smoothly.

Storage (SSD / HDD)

Where your games, files, and operating system are stored.

🛠️ Step-by-Step Build

1. Install CPU into motherboard

2. Add RAM sticks

3. Install SSD or HDD

4. Mount motherboard into case

5. Install GPU

6. Connect power supply cables

7. Boot and install Windows

⚡ Pro Beginner Tips

✔ Always check compatibility (CPU + motherboard)

✔ Don’t force components into place

✔ Use anti-static precautions

✔ Cable management improves airflow
